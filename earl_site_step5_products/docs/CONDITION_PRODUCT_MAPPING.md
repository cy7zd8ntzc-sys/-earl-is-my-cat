# Condition-to-Product Mapping

Use this as the public-page logic for supportive products.

| Condition | Owned supportive products | Affiliate / OTC categories | Prescription role |
| --- | --- | --- | --- |
| Acne | Gentle Cleanser, Oil-Free Moisturizer | OTC acne wash, sunscreen | Visit-dependent prescription acne plan |
| Rosacea/redness | Gentle Cleanser, Barrier Moisturizer, later Calm-Looking Serum | Mineral sunscreen | Visit-dependent diagnosis and Rx options |
| Melasma/dark spots | Gentle Cleanser, Barrier Moisturizer | Tinted mineral sunscreen | Visit-dependent pigment plan |
| Fine lines/aging | Gentle Cleanser, Barrier Moisturizer | Sunscreen | Visit-dependent retinoid-style plan |
| Hair loss | None initially | Topical minoxidil | Visit-dependent diagnosis and Rx options |
| Seb derm/dandruff | Gentle Cleanser for face support | Medicated dandruff shampoo | Visit-dependent topical Rx options |
| Sweating | None initially | Clinical-strength antiperspirant | Visit-dependent Rx options |
| KP/rough bumps | Rough-Feeling Skin Body Lotion | None initially | Usually supportive-first |
| Fungal/intertrigo | None initially | OTC antifungal when appropriate | Visit-dependent diagnosis/testing/Rx |
| Warts | None initially | OTC non-genital wart remover only | Visit-dependent, especially unclear lesions |
| Genital warts | None | None | Visit-dependent only |
| Herpes, scabies, impetigo, HS | None | Generally none on public page | Visit/triage-dependent |

Do not attach product cards to infection or STI pages unless there is a clear, non-misleading reason and appropriate warning language.
